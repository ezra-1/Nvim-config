## Requirement

- [Nerd Fonts][NerdFonts] Nerd Fonts is require if you want to display icons

- [Neovim 0.9.5+][Neovim] (not including nightly)

- [Pyvim][Pyvim] for python support

- [Rigrep][Rigrep] needed for telescope to work
